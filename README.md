# PF

React + Vite + TypeScript 기반 포트폴리오입니다.

## Scripts

```bash
npm install
npm run dev
npm run build
```

GitHub Pages 배포 경로는 `vite.config.ts`의 `base: '/PF/'`로 설정되어 있습니다.

## GitHub Pages

이 저장소는 `05solar/PF`의 프로젝트 페이지로 배포되도록 구성되어 있습니다.

- Vite base path: `/PF/`
- Build output: `dist`
- Deploy workflow: `.github/workflows/deploy.yml`
